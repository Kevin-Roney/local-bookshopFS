Local Bookstore API

Your one stop shop for the classics

Route Method Description Example 
/books
 GET Gets a list of all books with nested quotes List books 
/books/:id
 GET Returns the detail about the book matching the id in the parameter Get book 1 
/books
 POST Adds a new book 
curl -d  '{"title": "A Game of Thrones", "release": 1996}' -H 'Content-Type: application/json' 'https://kroney-local-bookshop-fs.herokuapp.com/books'
              
/authors
 GET Returns a list of all authors with nested books from that author List authors 
/authors/:id
 GET Returns the detail about the author matching the id in the parameter Get author 1 
/authors
 POST Adds a new author 
curl -d  '{"name": "George R.R. Martin", "dob": "1948-9-20", "pob": "Bayonne, New Jersey"}' -H 'Content-Type: application/json' 'https://kroney-local-bookshop-fs.herokuapp.com/authors'
